# FormSarpra-CRUD

### Deskripsi Aplikasi
Pada program berbasis Java (Desktop) ini mengambil tema Form Barang Siswa Tertinggal yang dikelola oleh SARPRA (Sarana dan Prasana) Sekolah. Pada form ini petugas harus melengkapi enam isian data siswa yang barangnya tertinggal.
